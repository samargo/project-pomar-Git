¿En qué consiste?

Sistema de gestión de usuarios para la Junta de Acción Comunal el Pomar

Funcionalidades

Realiza las funciones básicas de un CRUD (Create, Read, Update, Delete), es decir, permite crear, leer, editar y eliminar registros de una base de datos. Permite exportar la información completa o filtrada a un archivo Excel. Tiene paginación para mostrar de manera estructuruda y selectiva los registros. Además de permitir el ingreso, con nombre y contraseña de los usuarios.

Tecnologías

Para el desarrollo de la aplicación web se hizo uso de HTML5, CSS y Bootstrap, desde una perspectiva estética, y JavaScript y Node.js desde el enfoque funcional y lógico.
